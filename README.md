# Agent Skills

Reusable skills for AI coding agents (Claude Code, Cursor, Copilot, and [40+ others](https://agentskills.io)).

## Install

```bash
npx skills add ljclaeyssen/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [wowhead-tooltips](skills/wowhead-tooltips) | Integrate Wowhead item tooltips, icons, and colored links into any web project (Angular, React, Vue, vanilla) |
